자주 묻는 질문 FAQ

  • 1. 언어와 테마는 어떻게 변경하나요? SnapNote 설정의 화면 표시 관련 페이지에서 앱 언어와 라이트, 다크 또는 시스템 따르기 테마를 변경할 수 있습니다.
  • 2. iCloud 동기화는 무엇을 동기화하고 무엇은 동기화하지 않나요? 현재 버전은 제목, 본문, 태그, 즐겨찾기 상태, 보관 상태 등 텍스트 메모 관련 정보를 주로 동기화합니다. 첨부 파일은 기본적으로 기기에 남아 있으며 iCloud로 자동 업로드되지 않습니다.
  • 3. 왜 첨부 파일은 자동으로 동기화되지 않나요? SnapNote는 로컬 우선 앱입니다. 첨부 파일을 기기에 보관하면 불필요한 업로드를 줄이고 저장 공간 사용을 예측하기 쉬우며, 파일 위치를 더 명확히 관리할 수 있습니다.
  • 4. Face ID / Touch ID 앱 잠금은 어떻게 동작하나요? 설정에서 앱 잠금을 켜면 앱을 열 때 또는 사용자가 정한 재잠금 시점에 앱으로 돌아올 때 시스템 생체 인증을 요청합니다. 인증은 시스템이 처리하며 SnapNote는 생체 정보를 저장하지 않습니다.
  • 5. 로컬 데이터를 삭제하거나 앱을 제거하려면 어떻게 하나요? 앱 안에서 필요 없는 메모와 첨부 파일을 삭제할 수 있습니다. 기기에서 SnapNote를 제거하면 해당 기기의 로컬 데이터도 함께 삭제됩니다. iCloud 동기화를 켰던 경우 동기화된 텍스트 메모는 사용자가 삭제할 때까지 다른 동기화 기기나 iCloud 환경에 남아 있을 수 있습니다.
